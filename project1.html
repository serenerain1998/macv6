UX Process & Methodology

A comprehensive approach to medical software design

Research & Discovery

Conducted extensive UX research with orthopedic surgeons to understand surgical workflows, pain points, and ergonomic requirements. Analyzed existing arthroscopy systems and identified opportunities for design improvement.

  • Surgeon interviews and shadowing
  • Workflow analysis and journey mapping
  • Competitive UX analysis
  • Ergonomic design studies

Design & Interactive Prototyping

Designed comprehensive UI interfaces in Figma and created interactive prototypes using ProtoPie Studio. Integrated Arduino boards with medical cameras to create realistic data flow that was reflected in the UI designs displayed on HUDs in operating rooms.

  • Figma UI design and prototyping
  • ProtoPie Studio interactive prototypes
  • Arduino integration for real data flow
  • HUD interface design for operating rooms

Authentic UX Testing

Conducted comprehensive usability testing in real surgical environments with actual surgeons. Gathered detailed feedback on interface design, ergonomics, motion handling, and user experience during live procedures.

  • Live surgical environment UX testing
  • Surgeon feedback collection and analysis
  • Interface ergonomic evaluation
  • User experience motion analysis

Design Refinement & Launch

Refined the UI designs and interactive prototypes based on surgeon feedback. Collaborated with engineering teams to implement the final solution, achieving 100% surgeon adoption and 40% faster setup times.

  • UI design refinement
  • Prototype iteration
  • Engineering collaboration
  • User adoption monitoring
